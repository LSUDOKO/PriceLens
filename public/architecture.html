🔄 Data Flow
MCP Query Flow
User: "Find cheapest for MacBook Pro"
→ Claude formats MCP tool call
→ JSON-RPC over stdio
→ MCP Server compares sources
→ db-adapter queries store
→ Cheapest source returned
Web Query Flow
User visits /heatmap (Source Comparison)
→ fetch(/api/prices?type=all)
→ API Route handler queries db-adapter
→ Returns products + source prices
→ Browser renders comparison chart
Alert Flow
User sets alert via UI or MCP tool
→ POST /api/alerts stores in db-adapter
→ Alert saved to DynamoDB or memory
→ Appears in alerts dashboard
→ Queryable via MCP list_alerts tool
